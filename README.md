# Andrea Cappelluti — Portfolio

Personal portfolio website of **Andrea Cappelluti**, a Data Scientist and Master's student in Data Science and Management.

## About

A single-page portfolio showcasing my background, projects, skills, and contact information. Built with plain HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Structure

```
portfolio/
├── index.html   # Main page (sections: Home, About, Work, Education, Skills, Contact)
├── style.css    # Custom styles
└── app.js       # Interactive behaviour (navigation, animations, etc.)
```

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (Inter, JetBrains Mono)
- Font Awesome 6 (icons)

## Live

Open `index.html` in any browser or deploy to any static host (GitHub Pages, Netlify, Vercel).
